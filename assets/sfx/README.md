# Sound Effects

`attack-moves/` contains only the move sound effects referenced by BioMon.

Source collection: Pokemon SFX Attack Moves Collection by BellBlitzKing, updated 2020-03-16.

The game primarily uses Gen 5 Black/White attack sounds. Later-generation or custom BioMon moves were copied in under the in-game move name so `index.html` can load every move from one flat folder.
